xclip-daemon by rmdir

A tool for building klipper like with lite window managers

It's provides a toolkit independant daemon that listen on a local socket and collect selection buffer entries. You can communicate with the daemon via 2 commands : - get : returns stored lines - del : clear the buffers

Dependencies

It's only depends on xorg libraries and classic Unix stuffs has pthreads

Install

- get the sources - make && make install (or install xclipd wherever you want) - xclipd -n number of entries to store -s /path/to/the/socket.sock -d
